Programação em GoLang

Aprenda a programar em GoLang com este guia completo.

Histórico

O Go é uma linguagem de programação criada pelo Google em 2007 por Robert Griesemer, Rob Pike e Ken Thompson. A ideia por trás do Go era criar uma linguagem moderna, eficiente e fácil de aprender, que fosse adequada para a programação em sistemas distribuídos e em largaescala.

O Go foi lançado oficialmente em 2009 e teve sua primeira versão estável em 2012, e desde então tem se tornado cada vez mais popular na comunidade de desenvolvimento de software.

Características

  • Compilação rápida
  • Tipagem estática
  • Concorrência
  • Gerenciamento de memória automático
  • Simplicidade e clareza
  • Suporte para programação orientada a objetos

Instalação

Para instalar o Go em seu computador, siga os seguintes passos:

  1. Faça o download do instalador do Go em https://golang.org/dl/
  2. Execute o instalador e siga as instruções
  3. Configure as variáveis de ambiente
  4. Verifique se a instalação foi bem-sucedida executando o comando go version no terminal

Estruturas de controle

Go suporta as estruturas de controle básicas, como if, for, switch, e select.

// Exemplo de uso do if

if x > 0 {
    fmt.Println("x é positivo")
} 
else if x == 0 {
    fmt.Println("x é zero")
} 
else {
    fmt.Println("x é negativo")
}

// Exemplo de uso do for

for i := 0; i < 10; i++ {
    fmt.Println(i)
    if x > 0 {
        fmt.Println("x é positivo")
    } 
    else if x == 0 {
        fmt.Println("x é zero")
    } 
    else {
        fmt.Println("x é negativo")
    }
}
   
// Exemplo de uso do switch
switch diaDaSemana {
    case 1:
        fmt.Println("Domingo")
    case 2:
        fmt.Println("Segunda-feira")
    case 3:
        fmt.Println("Terça-feira")
    case 4:
        fmt.Println("Quarta-feira")
    case 5:
        fmt.Println("Quinta-feira")
    case 6:
        fmt.Println("Sexta-feira")
    case 7:
        fmt.Println("Sábado")
    default:
        fmt.Println("Dia da semana inválido")
}

Variáveis

Em Go, as variáveis são declaradas usando a palavra-chave var e o tipo da variável é especificado após o nome da variável.

// Exemplo de declaração de variável
var x int
var y float64
var z string

// Exemplo de atribuição de valor a uma variável
x = 10
y = 3.14
z = "Hello, world!"

// Exemplo de declaração e atribuição de valor a uma variável
var a int = 5
var b float64 = 2.5
var c string = "Olá, mundo!"

Operadores

Go suporta os operadores aritméticos básicos, como +, -, *, /, e %, além de operadores de comparação, como ==, !=, <, >, <=, e >=.

// Exemplo de uso de operadores aritméticos
var a int = 5
var b int = 2

var soma int = a + b
var subtracao int = a - b
var multiplicacao int = a * b
var divisao float64 = float64(a) / float64(b)
var resto int = a % b

// Exemplo de uso de operadores de comparação
var x int = 10
var y int = 5

var igual bool = x == y
var diferente bool = x != y
var maior bool = x > y
var menor bool = x < y
var maiorOuIgual bool = x >= y
var menorOuIgual bool = x <= y

Tipos de dados

Go possui diversos tipos de dados básicos, como inteiros, números de ponto flutuante, strings, booleanos, além de tipos compostos, como arrays, slices, maps, e structs.

// Exemplo de declaração de um array
var numeros [5]int

numeros[0] = 1
numeros[1] = 2
numeros[2] = 3
numeros[3] = 4
numeros[4] = 5

// Exemplo de declaração de um slice
var nomes []string

nomes = append(nomes, "João")
nomes = append(nomes, "Maria")
nomes = append(nomes, "José")

// Exemplo de declaração de um map
var idades map[string]int

idades = make(map[string]int)

idades["João"] = 20
idades["Maria"] = 25
idades["José"] = 30

// Exemplo de declaração de um struct
type Pessoa struct {
    nome string
    idade int
}

var pessoa1 Pessoa

pessoa1.nome = "João"
pessoa1.idade = 20

Funções

Em Go, as funções são declaradas usando a palavra-chave func, seguida do nome da função, dos parâmetros entre parênteses, e do tipo de retorno, caso haja.

// Exemplo de declaração de uma função que soma dois números inteiros
func soma(a int, b int) int {
    return a + b
}

// Exemplo de chamada de função
resultado := soma(5, 10)

Arrays

Em Go, um array é uma sequência fixa de elementos do mesmo tipo. O tamanho do array é especificado na declaração.

// Exemplo de declaração de um array
var numeros [5]int

numeros[0] = 1
numeros[1] = 2
numeros[2] = 3
numeros[3] = 4
numeros[4] = 5

// Exemplo de iteração sobre um array
for i := 0; i < len(numeros); i++ {
    fmt.Println(numeros[i])
}

Exemplos básicos de programação

A seguir, alguns exemplos básicos de programação em Go:

// Exemplo de programa que imprime a sequência de Fibonacci
func fibonacci(n int) []int {
    var resultado []int
    a := 0
    b := 1

    for i := 0; i < n; i++ {
        resultado = append(resultado, a)
        temp := a
        a = b
        b = temp + b
    }
    return resultado
}

fmt.Println(fibonacci(10))

// Exemplo de programa que calcula o fatorial de um número

func fatorial(n int) int {
    if n == 0 {
        return 1
    } 
    else {
        return n * fatorial(n-1)
    }
}

fmt.Println(fatorial(5))

// Exemplo de programa que verifica se um número é primo

func primo(n int) bool {
    if n < 2 {
        return false
    }
    for i := 2; i <= int(math.Sqrt(float64(n))); i++ {
        if n%i == 0 {
            return false
        }
    }
    return true
}

fmt.Println(primo(7))
